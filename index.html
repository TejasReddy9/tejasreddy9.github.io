Sentiment analysis on tweets

5th June 2017 by Tejas Reddy

Sentiment Analysis is the process of 'computationally’ determining whether a piece of writing is positive, negative or neutral. It’s also known as opinion mining, deriving the opinion or attitude of a speaker.

Requirements:

  1. Install Tweepy, a python client for the official Twitter API.
    pip install tweepy
  2. Install TextBlob, a python package for Natural Language Processing related to textual data.
    pip install textblob
  3. Get authentication code and API key after getting access to the Twitter REST API. Go to this link.
